============================
Installing async-mongoengine
============================

To use async-mongoengine, you will need to download `MongoDB <https://www.mongodb.com/>`_
and ensure it is running in an accessible location. async-mongoengine uses
Motor (the async MongoDB driver) under the hood, and all required dependencies
will be installed automatically.

async-mongoengine is available on PyPI, so you can use :program:`pip`:

.. code-block:: console

    $ python -m pip install async-mongoengine
